Discuss the scope of var, let, and const

var: Hoisting: Hoisted to the top of their containing function or global scope. Scope: Function-scoped. Reassignment: Can be reassigned.
let: Hoisting: Hoisted to the top of their containing block or function scope but not initialized until their declaration. Scope: Block-scoped. Reassignment: Can be reassigned.
const: Hoisting: Hoisted to the top of their containing block or function scope but not initialized until their declaration. Scope: Block-scoped. Reassignment: Cannot be reassigned but allows modification of object properties or array elements.

Tell the use cases of null and undefined

null: Indicates an intentional absence of value. Used for cleaning up references or as a default initialization.
undefined: Indicates the absence of a value due to the lack of assignment. Automatically assigned to uninitialized variables and missing object properties. Used for function return values and checking missing function arguments.

What do you mean by REST API?

REST API (Representational State Transfer Application Programming Interface) is an architectural approach for building web services. It uses standard HTTP methods, such as GET, POST, PUT, and DELETE, to interact with resources (e.g., data) on a server. REST APIs are known for their simplicity, statelessness, and uniform interface, making them widely used for web and mobile applications.
